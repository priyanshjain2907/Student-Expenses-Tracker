## 📊 Student Finance Tracker

A simple and efficient finance tracking application designed for students to manage their income, expenses, and savings. This project helps users gain better financial awareness and control over their daily spending habits.

---

## 🚀 Features

- 💰 Track Income and Expenses  
- 📅 Record transactions with date  
- 🏷️ Categorize expenses (food, travel, etc.)  
- 📊 View financial summary  
- 📁 Export data to CSV  
- 🗄️ SQLite database integration  

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Database:** SQLite  
- **Libraries Used:**  
  - sqlite3  
  - datetime  
  - csv  

---


## 🧠 How It Works

- The application stores income and expense data in a SQLite database.
- Users can:
  - Add income entries
  - Add expense entries with categories
  - View transaction history
- Data can be exported into CSV format for analysis.

---

## 📸 Example Use Cases

- Students tracking monthly allowance  
- Monitoring daily expenses  
- Budget planning
